📍 在 index.html 里怎么找位置？
1. 用文本编辑器打开 index.html
2. 搜索 const DATA = [
3. 找到你选择的分类（比如 id: 'ads'）
4. 在它的 items: [ 数组的最后一条后面，粘贴刚才复制的内容
5. 保存文件，刷新浏览器
工作流：
1. 打开 content_template.csv（Excel 会自动识别）
2. 在下面继续加行 → 保存为 CSV → 用 Excel 全选复制
3. 粘贴到下方文本框 → 点"生成 JSON" → 复制粘贴到 index.html
